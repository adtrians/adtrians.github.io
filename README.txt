
Adtrian — Film Editor portfolio (ready to deploy)

How to use
1. Extract the ZIP to your web server folder (public_html or www).
2. Open index.html to view the landing page. portfolio.html contains the grid with filters.
3. If you want the contact form to work, edit assets/php/contact.php and set $to to your email and deploy to a PHP-enabled server.
4. Images are loaded from picsum.photos as placeholders; replace assets/images or img src with your actual thumbnails.

Files included:
- index.html
- portfolio.html
- assets/css/style.css
- assets/js/script.js
- assets/php/contact.php

Enjoy — tweak fonts/colors as needed.
